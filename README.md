Visually exploring flows in weighted dynamic netowrks for data journalists

Installation
------------

```
git clone https://github.com/VALIDproject/valid.git
cd valid
npm install
```

Testing
-------

```
npm test
```

Building
--------

```
npm run build
```

Launching
---------

```
npm start
```
